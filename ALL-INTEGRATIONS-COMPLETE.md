## 🚀 How to Use

### Habit Scorecard
1. Open Habits module
2. Click "Scorecard" button or tab
3. Add habits and mark as +/-/=
4. Click "Improve" on bad habits to create new habit

### Phase Progression
1. Open any habit card
2. Click "Add Difficulty Progression"
3. Add phases (action + days)
4. Track progress, advance when ready

### Temptation Bundling
1. Open any habit card
2. Click "Add Temptation Bundling"
3. Fill need, want, and rule
4. Use suggestions or create custom

### Statement Builder
1. Open habit form
2. Fill in cue, action, location, identity
3. See real-time preview
4. Fill 4 Laws prompts
5. Use cue suggestions

### Envelope Budgeting
1. Open Expenses module
2. Click "💰 Allocate" button
3. Complete 5-step wizard
4. View envelope cards below stats

### Payment Modes
1. Open Expenses module
2. Click "💳 Modes" button
3. Add payment modes
4. View cards below envelopes
